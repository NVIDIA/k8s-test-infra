# Support

Community support for NVIDIA k8s-test-infra is provided on a best-effort basis.
The project does not offer a service-level agreement (SLA) or guaranteed response
time.

For installation, configuration, and usage questions, start a thread in
[GitHub Discussions](https://github.com/NVIDIA/k8s-test-infra/discussions).
Search existing discussions before posting and include the project version,
environment details, relevant configuration, and redacted logs.

Use the [issue chooser](https://github.com/NVIDIA/k8s-test-infra/issues/new/choose)
for reproducible bugs, feature requests, documentation problems, or a support
request that needs tracked project work. Do not report security vulnerabilities
in Discussions or public issues; follow the private reporting instructions in
[SECURITY.md](SECURITY.md).
